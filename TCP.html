TCP

TCP (Transmission Control Protocol) is a fundamental communication protocol in the Internet protocol suite. It provides reliable, ordered, and error-checked delivery of data between applications running on hosts communicating via an IP network.

TCP operates at the transport layer of the OSI model and is responsible for establishing and maintaining a connection between two devices. It ensures that data packets are transmitted in the correct order and that any lost packets are retransmitted.
