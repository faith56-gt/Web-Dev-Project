DNS

DNS (Domain Name System) is a hierarchical and decentralized naming system for computers, services, or other resources connected to the Internet or a private network. Translates human readable domain names into machine-readable IP addresses.

  1. User Types URL in Browser
  2. Browser Checks local DNS Cache
  3. DNS Query is Sent
  4. DNS Server Responds with IP Address
  5. Browser Connects to Web Server using IP Address
